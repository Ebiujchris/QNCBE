# 🚀 Quick Start Guide

Your Neon database is already configured! Follow these simple steps:

## 1. Install Dependencies
```bash
npm install
```

## 2. Setup Database
```bash
npm run setup-db
```

## 3. Start Backend Server
```bash
npm run dev
```

## 4. Start Frontend (New Terminal)
```bash
cd ../QNCFE
npm install
npm run dev
```

## 5. Open Browser
Visit: http://localhost:3000

## 6. Login as Admin
- Email: admin@qnc.com
- Password: admin123
